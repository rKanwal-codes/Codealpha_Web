# Age Calculator

## 📌 Project Overview
Age Calculator is a simple web application built using HTML, CSS, and JavaScript. It allows users to enter their date of birth and calculates their exact age in years, months, and days.

This project was developed as part of the CodeAlpha Web Development Internship.

---

## 🚀 Features

- Enter Date of Birth using a date picker
- Calculate age instantly
- Display age in:
  - Years
  - Months
  - Days
- Input validation for empty fields
- Simple and user-friendly interface
- Responsive design

---

1. Open the folder in VS Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.

---

## 📖 How It Works

1. User selects their Date of Birth.
2. JavaScript retrieves the selected date.
3. The current date is obtained using the JavaScript Date object.
4. The difference between the current date and birth date is calculated.
5. The age is displayed in years, months, and days.
